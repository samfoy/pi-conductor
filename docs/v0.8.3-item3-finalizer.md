### PRD update content (`2e4ea18`)

A consolidated decision-log entry dated 2026-05-19 enumerates three architectural
conventions established by the redesign:

1. **Component-layer styling (Option A)** — pure renderers stay monochrome; ANSI lives
   in `transcript-style.ts` invoked from the Component layer. Future styling work follows
   this pattern.
2. **Thinking-block summary by default + `t` toggle** — refines the original v0.4
   thinking-toggleable design at PRD line 285.
3. **`Run.lastEventAt` convention** — every event-handler push to `run.messages` writes
   `lastEventAt = Date.now()`; `renderHeader` consumes it for live activity. Future
   header live-status work reads from this field.

Width-primitive migration (`6793a5a`) is also cited as the off-script pre-fix that
pre-paid debt for the Component-layer fork.
